## LogAp Lab - Visão Computacional

### Requerimentos

- Python 3.9 ou superior

### Instalação

Crie um ambiente virtual:

```
python -m venv venv
```

Ative-o:

```
source ./venv/bin/activate # Linux e MacOS
.\venv\Scripts\activate.bat # Windows
```

Instale as dependências com:

```
pip install -r requirements.txt
```

### Execução do algoritmo de Viola-Jones

```
python ./viola-jones/main.py # Linux e MacOS
python .\viola-jones\main.py # Windows
```
